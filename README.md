# 年终汇报项目 2025

一个类似PPT的滚动翻页年终汇报项目，参考现代UI设计风格。

## 功能特点

- ✨ **流畅的滚动翻页**：类似PPT的页面切换效果
- 🎨 **现代UI设计**：参考图片风格，使用渐变色彩和卡片式布局
- 📱 **响应式设计**：支持桌面和移动设备
- ⌨️ **键盘导航**：支持方向键、PageUp/PageDown、Home/End键
- 🖱️ **鼠标滚轮**：支持鼠标滚轮翻页
- 📱 **触摸支持**：移动端支持滑动翻页
- 🎯 **导航指示器**：右侧显示当前页面位置

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 使用以下方式翻页：
   - **鼠标滚轮**：滚动鼠标滚轮
   - **键盘**：
     - `↓` 或 `PageDown`：下一页
     - `↑` 或 `PageUp`：上一页
     - `Home`：第一页
     - `End`：最后一页
   - **触摸**：在移动设备上上下滑动
   - **导航点**：点击右侧的导航点直接跳转

## 自定义内容

### 修改幻灯片内容

编辑 `index.html` 文件中的各个 `<section class="slide">` 部分，修改文字、数据等内容。

### 修改样式

编辑 `styles.css` 文件，可以修改：
- 颜色主题（CSS变量在 `:root` 中定义）
- 字体大小和样式
- 布局和间距
- 动画效果

### 添加/删除幻灯片

1. 在 `index.html` 中添加或删除 `<section class="slide">` 元素
2. 在 `script.js` 中修改 `totalSlides` 变量的值

## 浏览器支持

- Chrome（推荐）
- Firefox
- Safari
- Edge

## 注意事项

- 建议使用现代浏览器以获得最佳体验
- 首次加载时页面会自动滚动到顶部
- 滚动动画有1秒的防抖时间，避免快速滚动造成的问题
